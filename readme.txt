== Description ==

This GTG Product Block is one of the most powerful plugin for Gutenberg that is compatible with WooCommerce to display your products on posts and pages. Using this Free Gutenberg plugin for Products, you can create beautiful features for product with lookbook collection. It's possible to add multiple images into lookbook and configure for block. Set Overlay, Point 1, point 2, ....

For more information on what blocks are available, and how to use them, check out the official documentation: https://wpdocs.gitbook.io/jenik/settings/block/product-lookbook

Use this plugin if you want create the lookbook product for WooCommerce.

- **Lookbook Products**

== Getting Started ==

= Minimum Requirements =

* WordPress 5.0 or greater
* WooCommerce 4.2 or greater
* PHP version PHP 7.2 or greater
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)

Visit the [documentation](https://docs.gutengeek.com/?utm_source=wporg-repo&utm_medium=link&utm_campaign=readme) for a detailed list of server requirements.

== Installation ==

GTG Product Blocks works with WordPress version 5.0 or higher.
You can install GTG Product Blocks for WooCommerce Blocks via WordPress plugin directory or upload files to your server at wp-content/plugins

Refer to our [getting started article](https://docs.gutengeek.com/topics/getting-started/) to know more.

For more information, please visit our [Knowledge Base](https://docs.gutengeek.com/?utm_source=wporg-repo&utm_medium=link&utm_campaign=readme")
